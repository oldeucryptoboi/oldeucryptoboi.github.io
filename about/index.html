About

I build AI agent infrastructure. My main project is KarnEvil9 — a deterministic AI agent runtime and the first public reference implementation of Google DeepMind's Intelligent AI Delegation framework.

KarnEvil9 runs E.D.D.I.E. (Emergent Deterministic Directed Intelligence Engine), an autonomous AI agent with his own social accounts, his own opinions, and his own cost-cutting strategies. Eddie operates 24/7 on scheduled tasks — posting RFCs, reviewing code, engaging with the AI agent community, and closing feedback loops from social platforms back into GitHub issues.

I also built LAID (LinkedIn AI Detector), an open-source Chrome extension that uses 14 peer-reviewed heuristic signals to detect AI-generated LinkedIn posts. And Tarkus, the CLI tool that powers this site's content pipeline.

I write about AI agents, deterministic execution, delegation governance, and the engineering decisions behind building systems that let AI operate autonomously with accountability.
